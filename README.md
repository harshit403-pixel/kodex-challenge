# kodex-challenge

# 🎬 Pixar Animation Studios – Frontend Reimagine

A visually rich, fully responsive frontend website inspired by Pixar Animation Studios, built using pure HTML and CSS with modern layouts and smooth UI interactions.

# 🌐 Live Demo
👉 Live Website:
https://kodex-challenge.vercel.app/

# ✨ Features
Sticky hero section with cinematic background overlay
Fixed header with blur effect and smooth transitions
Interactive “Coming Soon” carousel using CSS hover expansion
Careers section with sticky layout
Advanced gallery grid
Masonry-style layout on desktop
Horizontal scroll with snap on mobile
Fixed footer reveal effect
Fully responsive design (Desktop, Tablet, Mobile)
Custom fonts and smooth hover animations

#🛠️ Tech Stack

HTML5
CSS3
Flexbox and CSS Grid
position: sticky and position: fixed
Media queries
Hover-based interactions
Google Fonts
Custom font (Explorer Condensed)
No JavaScript used — all interactions are CSS-driven.

# 📱 Responsive Design

Desktop: Full layout with sticky sections and complex grid
Tablet: Simplified layout and reduced grids
Mobile: Single-column layout with scrollable gallery

# 🧠 Key Concepts Used

CSS reset and base styling
Sticky and fixed positioning
Layered gradients and overlays
Aspect-ratio handling
Scroll-based visual hierarchy
Hover-driven layout transitions

# 🚀 How to Run Locally

Clone or download the repository
Open index.html in any modern browser
No dependencies or build tools required

🎯 Purpose

This project was created to practice advanced CSS layouts, explore sticky scrolling effects, and build a production-style landing page without using JavaScript.
